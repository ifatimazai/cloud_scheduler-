🔒 THREAD SAFETY
═══════════════════════════════════════════════════════════════════════════

All shared resources are protected:

  Resource Pool:
    pthread_mutex_t resource_lock;
    Protects: CPU, memory, storage allocation

  Deadlock Detector:
    pthread_mutex_t detector_lock;
    Protects: Resource request tracking

  Simulation Context:
    pthread_mutex_t sim_lock;
    Protects: VM queues, shared state

  Synchronization Pattern:
    • Lock before critical section
    • Perform operation atomically
    • Unlock immediately after
    • No nested locks (prevents deadlock)
